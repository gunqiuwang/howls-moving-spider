## 2. 当前项目方向

当前项目已经可以在线运行。核心玩法方向是：

> 八足机械移动城堡探索小游戏。

当前不是电影原版双足城堡路线。

本轮目标不是增加新玩法，而是做一轮视觉气质改造，让项目更接近宫崎骏 / 吉卜力式童话机械世界。

关键词：

- 温暖
- 手工感
- 旧机械
- 生活气
- 风与光
- 童话山野
- 会走路的房子群

---

## 3. 必须遵守

1. 保留 8 足 IK 行走。
2. 保留飞行模式。
3. 保留金币收集。
4. 不改成双足结构。
5. 不做赛博朋克、军武机甲、冷科幻。
6. 不复刻具体电影镜头、角色或资产。
7. 不引入大型外部 3D 模型。
8. 优先做高收益低风险改动。

---

## 4. 推荐改造顺序

1. 光照、天空、雾气。
2. 城堡主体轮廓，让它更像会走的房子群。
3. 材质统一：旧铁皮、老木头、暗红瓦、暖窗光。
4. 环境童话感：草地、云、村庄、风车、水井、围栏。
5. UI 风格统一：从调试 HUD 改成童话机械仪表。

---

## 5. 每轮输出要求

每轮改完后输出：

- 修改文件列表。
- 每个文件改动目的。
- 视觉变化说明。
- 是否影响玩法。
- `npm run build` 结果。
- 浏览器控制台是否还有 error。
- 还剩哪些问题。

验收以 `docs/ACCEPTANCE.md` 为准。
