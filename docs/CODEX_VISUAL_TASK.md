# Codex 视觉改造任务单

> 目标：让项目更符合“宫崎骏 / 吉卜力式童话机械世界”的气质。
> 范围：视觉和氛围优先，不做玩法扩展，不推翻 8 足 IK。

---

## 0. 开工前必读

Codex 开工前请先阅读：

1. `README.md`
2. `docs/VISUAL_DIRECTION.md`
3. `docs/ACCEPTANCE.md`
4. `docs/QA_ONLINE_CHECK_2026_06_27.md`
5. `docs/REDESIGN.md` 只作为未来备选，不进入当前主线

---

## 1. 总任务

请基于现有代码做一轮 **视觉气质优先** 改造。

一句话目标：

> 保留当前八足移动城堡玩法，但让它更像活在宫崎骏世界里的旧机械童话城堡。

不要把它改成双足电影原版，也不要做成赛博朋克、军武机甲或冷硬科幻。

---

## 2. P0：第一轮必须优先做

### 任务 1：光照与天空重调

涉及文件优先看：

- `src/world/World.js`
- `src/materials.js`
- `src/ui/styles.css`

目标：

- 画面更温暖、更柔和。
- 天空更有童话黄昏 / 清晨感。
- 雾气更像空气透视，不要脏灰。
- 阴影不要太硬。

验收：

- 场景不再像默认 3D demo。
- 草地、城堡、天空颜色关系更协调。
- 远景有空气感。

---

### 任务 2：强化移动城堡轮廓

优先看：

- `src/spider/Spider.js`
- `src/materials.js`

目标：

- 第一眼更像“会走的房子群”。
- 屋顶、烟囱、窗户、木梁、露台更明显。
- 机械腿保留，但不要压过城堡主体。

建议：

- 增加或强化暖窗光。
- 增加房屋堆叠差异。
- 增加旧木梁、补丁铁皮、烟囱等识别点。
- 让正面“脸感”更友好，减少恐怖感。

验收：

- 远看仍能识别为移动城堡。
- 不只是蜘蛛机器人。

---

### 任务 3：统一材质语言

优先看：

- `src/materials.js`
- `src/textures.js`
- `src/spider/Spider.js`
- `src/world/World.js`

目标：

- 锈铁更暖。
- 木头更旧。
- 瓦片更柔和。
- 石头更自然。
- 窗光更有炉火感。

验收：

- 材质之间不打架。
- 没有现代塑料感和冷金属感。

---

## 3. P1：P0 完成后再做

### 任务 4：增加生活气细节

方向：

- 更多暖窗光。
- 更柔和的烟囱蒸汽。
- 帘布、旗子、挂件、外露木梁。
- 小灯、黄铜件、补丁铁皮。

目标：

- 城堡看起来像有人住，而不是空壳机械。

---

### 任务 5：优化环境童话感

优先看：

- `src/world/World.js`
- `src/world/terrain.js`

方向：

- 草地更自然。
- 云更柔软。
- 村庄、风车、水井、围栏更绘本化。
- 路径与场景摆放更像有生活动线。

目标：

- 世界不是测试场，而是童话山野。

---

### 任务 6：UI 世界观统一

优先看：

- `index.html`
- `src/ui/styles.css`
- `src/main.js`

方向：

- HUD 像黄铜铭牌或旧机械仪表。
- 任务提示像冒险目标。
- 按钮更有童话机械感。
- 中文文案保持统一。

目标：

- UI 不再像调试面板。

---

## 4. 禁区

不要做：

1. 不要把 8 足改成 2 足。
2. 不要复刻电影具体画面、角色、资产。
3. 不要做赛博朋克、军武机甲、冷科幻。
4. 不要引入大型外部 3D 模型。
5. 不要在本轮增加多人、剧情、升级系统。
6. 不要为了细节破坏移动端性能。

---

## 5. 建议实施顺序

1. 修依赖，确保本地能 build。
2. 光照 / 天空 / 雾。
3. 城堡主体轮廓。
4. 材质统一。
5. 环境童话感。
6. UI 风格统一。
7. 浏览器复查控制台。
8. 截图或记录视觉变化。

---

## 6. 每轮输出要求

每轮完成后请输出：

- 修改文件列表。
- 每个文件改动目的。
- 视觉变化说明。
- 是否影响玩法。
- `npm run build` 结果。
- 浏览器控制台是否有 error。
- 还剩哪些视觉问题。

---

## 7. 本轮验收问题

本轮完成后至少回答这些问题：

1. 第一眼更像移动城堡了吗？
2. 它更温暖、更童话了吗？
3. 它有旧机械和生活气了吗？
4. 环境是否更像宫崎骏式山野世界？
5. UI 是否更贴合这个世界？
6. 行走、飞行、金币收集是否仍然正常？
