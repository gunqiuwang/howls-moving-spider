## 设计风格

### 视觉风格：吉卜力蒸汽朋克

- **城堡**：锈蚀铁皮 + 旧木板 + 红棕瓦片，铆钉纹理，不规则堆叠
- **材质**：MeshStandardMaterial，低金属度（0.15-0.2），高粗糙度（0.85-0.9）
- **色调**：暖棕/锈铁/暗金，参考色板 `GHIBLI`（materials.js）
- **光照**：半球光（天蓝 + 暖棕地）+ 金色主光 + ACES 色调映射（曝光 0.95）
- **氛围**：FogExp2 指数雾（密度 0.004），梦幻感

| 用途 | 色值 | 说明 |

| 天空 | `#1a3a6a` → `#f0c878` | 深蓝到金色地平线渐变 |

| 城堡 | `#6a5038` | 锈铁底色（带铆钉纹理） |

## 核心系统

### IK 行走系统

- 8 条腿，每条 `femur` + `tibia` 两段骨骼
- 交替三角步态（同一时间最多 4 条腿离地）
- 闭式两骨 IK 求解（支持 decomposed 和 analytic 两种模式）
- 膝盖摆动搜索（TUNE.aRange / aSamp）避开碰撞
- 身体跟随地形倾斜（bodyConform + feetN）

### 飞行模式

- F 键或触屏"起飞"按钮切换
- 飞行时 `this.up = Y_UP`（保持水平）
- 腿向外展开如翅膀（spread 1.8x，sweep 0.3）
- 降落时自动检测地面，平滑着陆

### 碰撞系统

- 球体/盒体射线求交（闭式）
- 地形步进光线（ray-march + 二分法）
- 风车、村庄房屋均有碰撞箱（可攀爬）

## 开发指南

```bash
# 安装依赖
npm install

# 本地开发（热重载）
npm run dev          # 默认端口 5173

# 构建
npm run build        # 输出到 dist/

# 本地预览构建产物
npx serve dist       # 或 python3 -m http.server 8093 -d dist

# 部署
git push             # Vercel 自动部署
```

### 关键设计约束

1. **保留 8 条腿 IK 系统** — 不修改骨骼和运动逻辑
2. **低多边形** — 纯代码建模，不导入外部资产（当前阶段）
3. **性能优先** — 移动端要能跑 30fps
4. **全中文 UI** — 所有界面文字用中文
